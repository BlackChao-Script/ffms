# ffms

家庭成员管理系统(主要是帮朋友搭建的毕业设计)

ffms = Spring Boot后端(朋友设计) + Vue前端

后端代码地址：[码云](https://gitee.com/zhike-violet/ffms.git)

## 前端技术栈

> - Vue 3(setup 语法糖)
>   
> - pinia
>   
> - Vue-router
>   
> - vite
>   
> - Ts
>   
> - Element-Plus
>   
> - Less
>   
> - Axios
>   

## 启动

1. 需要配置 [Nodejs](https://nodejs.org/en/download/) 环境
  
2. 数据库导入sql/ffms.sql下的数据库文件
  
3. 启动管理系统
  
  ```
  npm install
  npm run dev
  ```
